# Banking Project - Selenium BDD Framework

This project is a Selenium Automation Framework using:

- Java
- Selenium WebDriver
- Cucumber BDD
- TestNG
- Page Object Model
- Maven

## Features
- Login functionality testing
- Customer operations
- Manager operations(Multiple users)
- Transaction verification
-Extent reporting 
## Framework Design
- Page Object Model
- Data driven testing
- Cucumber BDD feature files

## Tools Used
- Eclipse
- GitHub
- Maven
- Selenium
- Cucumber
- TestNG
